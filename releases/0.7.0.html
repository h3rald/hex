v0.7.0 — 2026-03-04

Fixes

  • Fixed several memory leaks in the {{sym-if}}, {{sym-try}}, {{sym-error}}, {{sym-args}}, {{sym-get}}, and {{sym-run}} symbols.
  • Fixed memory leaks in bytecode interpretation of integers, strings, and native symbols.
  • Fixed memory leaks in CLI argument handling.
  • Fixed a potential crash in {{sym-==}} when comparing symbols with non-symbol types.
  • Fixed the {{sym-run}} symbol to use dynamically allocated buffers instead of fixed-size buffers for command output.
  • Fixed bytecode string length decoding loop.
  • Fixed bytecode user symbol index to use 2-byte little-endian encoding.

Chores

  • Increased test coverage.
