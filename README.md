# hex

_hex_ is a tiny, minimalist, slightly-esoteric, concatenative programming language.

```
   _*_ _
  / \hex\*
 *\_/_/_/
      *
```

&rarr; For more information, see [the hex web site](https://hex.2c.fyi).
